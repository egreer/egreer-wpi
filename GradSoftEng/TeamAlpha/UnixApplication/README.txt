In order to run the Unix Tests application you must have these components installed:
Tester
Suite Runner
Starter
Tail
Head
Uniq
Cat
Sort  


The application can be customized by removing any of the components 
that do not need to be tested, or testing them individualy.

Note:  running this application will throw an exception about not being able to
 activate starter, this is necessary to ensure proper shutdown for EclEmma coverage.
